Microsoft Visual Studio

Visual Studio (current version Visual Studio 2022, which is 64-bit) is Microsoft's premier IDE for Windows and macOS. With Visual Studio, you can develop, analyze, debug, test, collaborate on, and deploy your software.

Download & Install

  1. Click on this link
  2. Choose the community 2022 or VS for Mac

  3. Open up Visual Studio Installer and choose this, then click Install

  4. Once the download is done, you will be asked to reboot the PC to complete Visual Studio setup

  5. Post reboot, open the Visual Studio IDE

    Select a theme of your choice

    Click Start Visual Studio

  6. That's it. Now you are ready to develop Applications using C/C++ in Visual Studio. In Visual Studio IDE, you can create new applications by clicking on Create a new project option as shown in the below image.
